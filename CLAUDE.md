# S2K «Трассировка» — премиум-редизайн (контекст проекта)

> Новый сайт студии S2K Studio уровня Awwwards. Арт-направление «ТРАССИРОВКА»: тёмное поле,
> сетка-оси, сигнатурная непрерывная линия-«магистраль» через всю страницу.
> Это **6-е, отдельное** направление — не путать с `s2k-new` (кобальт, 5 вариантов) и
> легаси `s2k-main`. Собираем с нуля в этой папке (`s2kprem`).

## Процесс
Двухэтапный. **Сейчас: конец Этапа 1 (анализ + арт-предложение) → КОНТРОЛЬНАЯ ТОЧКА.**
Код сайта не писать до явного «ок» по арт-направлению.

## Главный констрейнт
**Медиа-голод:** ни одного реального кадра по кейсам; фото команды — заглушка. Сток и
генерации запрещены. Арт-направление «Трассировка» (типографика + сетка + магистраль)
выбрано в том числе потому, что **держится на типографике, а не на фото** — медиа-голод его
не ломает. Кейсы строим как типографические паспорта; полноэкранные медиа-герои — в TODO.

## Планируемый стек (Этап 2)
Next.js 15 (App Router) + TS strict · Tailwind v4 (@theme токены) · Framer Motion · Lenis ·
next/font (Geologica + Onest + JetBrains Mono, subset latin+cyrillic) · next/image · next/og.
Без three.js/GSAP/слайдер-китов. Магистраль — чистый SVG path-drawing (stroke-dashoffset),
per-section пути, IntersectionObserver, reduced-motion → статика.

## Решения (контрольная точка пройдена, 2026-06-13)
- **Палитра:** чёрно-оранжевая. Акцент/магистраль — `#FF4D00` (`--accent`), мелкий текст —
  `--accent-soft #FF6A2B`, инверсные секции — `--accent-ink #B53800`. Фон `#121212`.
  Кобальт отвергнут — осознанный ребрендинг акцента.
- **Магистраль:** v3 — **«ОСЕВАЯ ШИНА · Живой ток / CHARGE-TO-CONTACT»** (2026-06-13, по
  фидбеку «линия должна сама двигаться и вести к продаже»). Геометрия v2 сохранена (центральная
  ось + «карманы» вокруг заголовков), но переработаны ДВЕ вещи:
  1) **Автономное движение** — пакет (бело-горячая голова + хвост-комета) **непрерывно едет ВНИЗ
     сам, без скролла**: чистый CSS `@keyframes trace-flow` по `stroke-dashoffset` 1→0 (ноль
     React-ре-рендеров/кадр). Скролл лишь модулирует `--boost` (opacity bloom + толщина головы) на
     непересекающихся с dashoffset свойствах — нет гонки. Проверено замером: dashoffset меняется
     без скролла.
  2) **Крещендо к продаже** — `heat` из `index` (01→0.15 … 06→1): к нижним секциям ток **быстрее**
     (`--flow-dur` 3.6s→2.0s), **горячее** (color-mix к акценту) и **плотнее** (2 пакета у TechStack/
     Contact). Стаггер `--flow-delay` сшивает посекционные пакеты в одну «реку» вниз. Узлы вспыхивают
     по тому же CSS-такту. Contact — **терминал-маяк** (`.trace-beacon`, постоянно зажжён + дыхание).
     CTA «Обсудить проект» — заряд-ореол `.trace-cta-charge` (энергия «прибывает» к кнопке).
  Reduced-motion — статичный зажжённый ток; мобайл — прямой центр-ствол с тем же бегущим пакетом,
  без фильтров. API `TraceSegment` сохранён байт-в-байт (6 секций не тронуты). Выбрано воркфлоу
  (charge-to-cta из 3 концепций). Файлы: `Trace.tsx` + добавки в `globals.css`. Прод-сборка зелёная.
